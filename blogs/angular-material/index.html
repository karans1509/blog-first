Incorporating Angular Material into Angular app

Feb 9, 2018

While developing the front-end for a web app, we might realize that we are using certain UI components quite often. And it's really time consuming to build those components from scratch everytime we require them. Another thing we might notice is that they might look different on different screens, or different browsers. To solve this issue, we are going to look at Angular Material, which is google's material design library developed specifically for Angular.

Angular material library provides us with ready-to-use components that will work across different browsers and screen sizes. They also provide a lot of themes to choose from, which you can customize accordingly. This library was developed to be used seamlessly with the Angular framework. Next, we will see how to actually use it in a project. For the following demo I am assuming that you have basic knowledge of the Angular framework.
